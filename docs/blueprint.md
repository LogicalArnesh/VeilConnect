# **App Name**: VeilConnect

## Core Features:

- Secure User Authentication: Comprehensive user authentication system including User ID/passcode login, new user registration with email verification, forgot password functionality, and integrated Google authentication. After successful passcode login, a 6-digit security key will be sent via email from a designated address for two-factor authentication.
- Role-Based Access Control: Manage user roles (Admin/User) and display the assigned role prominently next to the user's name on the dashboard, dictating access to different modules and data visibility.
- Personalized User Dashboard: A dynamic dashboard displaying the user's ID and role in the top right corner, along with the current time, day, and date. Provides personalized access to tasks, meetings, schedules, and progress modules based on the user's assigned role.
- Task Management Module: Allows users to create, view, update, and manage their personal tasks. Admins have a comprehensive view to oversee and manage tasks for the entire team, with configurable sample tasks provided.
- Meeting Management Module: Facilitates the scheduling, viewing, and tracking of team meetings. Users can see their relevant meetings, while Admins can manage all team meetings and their details.
- Team Progress Tracking: Provides an overview of individual and team progress. Users can update their own progress, while Admins gain access to detailed progress reports across the entire team.
- AI-Powered Content Suggestions Tool: Leverages generative AI to suggest and provide templates for common team activities, such as task descriptions, meeting agenda points, or structured progress report outlines, aiding admins in setting up activities efficiently.

## Style Guidelines:

- The app features a sophisticated dark color scheme, aligning with the 'VEIL CONFESSIONS' brand identity for professionalism and confidentiality.
- Primary color: Deep, muted indigo (#5859AC) for core UI elements, selected for its depth and professional tone. This color signifies reliability and quiet authority.
- Background color: A very dark, desaturated blue-grey (#16171C), providing a clean and non-distracting canvas that ensures good readability for lighter text and UI components.
- Accent color: A vibrant cyan-blue (#68CBF2), offering strong contrast for interactive elements, highlights, and important notifications to draw user attention effectively on the dark background.
- Headline and body font: 'Inter', a modern sans-serif typeface, chosen for its excellent readability, clean lines, and neutral, professional aesthetic suitable for a data-rich dashboard.
- Modern, crisp line icons that maintain a clean and professional look, enhancing clarity and user experience without overwhelming the UI. Icons should subtly reflect the themes of team collaboration, security, and organization.
- The login screen will prominently display the 'VEIL CONFESSIONS' team logo and name, followed by clear input fields. The dashboard will feature a structured, modular layout to clearly delineate tasks, meetings, schedules, and progress sections, adapting for role-based content. The user ID and role will be fixed at the top right, while time and date elements will be strategically placed for quick reference.
- Subtle and functional animations for user feedback, such as transitions upon login success, task completion, or data loading, to provide a smooth and polished user experience without distractions.
